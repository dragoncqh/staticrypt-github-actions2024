Inserting Images

  1. Select one of the following:
    1. Insert > Graphic menu choice.
    2. Insert Graphic toolbar button.
  2. The Resource Manager opens with the Image tab active.
  3. Browse for the image you want to insert.
  4. Use the Type field to filter the graphics displayed in the Resource Manager browser.
  5. Use the Insert drop-down list to select the element to insert into your document.
  6. Press the Insert button.

    This inserts a reference to the graphic into the document you are editing.

    If the current context in your document is not a valid insertion point, this button is disabled.
