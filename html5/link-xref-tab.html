Inserting Links and Cross References

The Link/Xref tab has two different states for inserting links or cross references:
  • local documents
  • documents on the Internet

Inserting links or cross references to local documents

  1. Select one of the following:
    1. Insert > Link/Xref menu choice.
    2. Link toolbar button.
  2. The Resource Manager opens with the Link/Xref tab active.
  3. Browse for the link or cross reference you want to insert.
  4. The Type field determines the type of documents displayed in the Resource Manager browser.
    Use this drop-down list to filter the documents displayed in the browser by the type of document. You can choose the following types of documents:
    • Topic — Only display DITA topic documents.
    • Topic or Map — Display either DITA map or topic documents.
    • Map — Only display DITA map documents.
    • Image — Only display graphic files.
    • PDF — Only display PDF documents.
    • HTML — Only display HTML documents
    • Any — Display all types of documents.
  5. Use the Show drop-down list to filter the elements displayed for documents in the browser by the type of element.
  6. Use the Insert drop-down list to select the element to insert into your document.
  7. Press the Insert button.

    This inserts a link or cross reference to the target document into the document you are editing.

    If the current context in your document is not a valid insertion point, this button is disabled.

Inserting links or cross references to documents on the Internet

For inserting links or cross references to a document on the Internet, the Link/Xref tab contains the following options:
  1. Select one of the following:
    1. Insert > Link/Xref menu choice
    2. Link toolbar button
  2. The Resource Manager opens with the Link/Xref tab active.
  3. Press the Browse the Web button on the Link/Xref tab.

    This changes to the state for inserting references to the Internet and invokes a web browser.

    If you already have a web browser open, the browser is made the active window. If you do not have a browser open, your default web browser is invoked.

  4. Use the web browser to find the URL you want to insert and then go back to the Link/Xref tab.
  5. The URL field displays the URL to be inserted for your link or cross reference.

    By default, this field displays the URL for the document currently open in your active web browser. The URL field automatically updates when you browse to a new web page and move the focus back to the Resource Manager. You can also edit this field.

  6. Press the Insert button.

    This inserts a link or cross reference to the target URL into the document you are editing.

    If the current context in your document is not a valid insertion point, this button is disabled.
