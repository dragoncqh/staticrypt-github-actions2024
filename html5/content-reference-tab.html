Inserting Content References

  1. Select the Insert > Content Reference menu choice.
  2. The Resource Manager opens with the Content Reference tab active.
  3. Browse for the content reference that you want to insert.
  4. The Type field determines the type of document displayed in the Resource Manager browser.
    Use this drop-down list to filter the documents displayed in the browser by the type of document. You can choose the following types of documents:
    • Map - Only display DITA map documents.
    • Map or Topic - Display either DITA map or topic documents.
    • Topic - Only display DITA topic documents.
    • Any - Display all types of documents.
  5. Use the Show drop-down list to filter the elements displayed for documents in the browser by the type of element.

    The contents of this list varies based on the type of document you are editing.

  6. Press the Insert button.

    This insert a content reference to the target element into the document you are editing.

    If the current context in your document is not a valid insertion point, this button is disabled.

    Note:

    If the element you are inserting as a content reference does not currently have an ID assigned to it, an ID is generated for the element if you have write access to the source file.
