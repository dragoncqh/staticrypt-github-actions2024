入门基础教程

准备工作

以sqlite为例，在windows平台，用VS2005/VC8来搭建开发环境。(别的平台或别的数据库下载相应的库文件)

  1. 下载ODB编译器安装包：odb-2.4.0-i686-windows.zip（生成持久化对象的SQl和头文件，用于持久化相关）

  2. 下载ODB核心库：libodb-2.4.0 libodb-2.4.0.zip

  3. 下载数据库运行库libodb-sqlite-2.4.0.zip,odb操作sqlite的相关功能，编译出来的odb-sqlite-d.lib(Debug版本)在examples中需要

  4. 下载例子程序odb-examples--2.4.0.zip,odb的例子和演示

  5. 下载sqlite的源文件(用sqlite所以需要)

  1. 安装ODB编译器：
    1. 解压odb-2.4.0-i686-windows到目录D:\user\desktop\C++\ODB C++ORM\
    2. 把bin目录加到环境变量Path中：D:\user\desktop\C++\ODB C++ORM\odb-2.4.0-i686-windows\bin
      注：
      不能仅复制odb-2.4.0-i686-windows\bin\odb.exe到C:\Windows\System32目录下而不修改环境变量，无效。
    3. 命令行输入odb --version看是否安装成功， 如现如下提示，可以就说安装成功了 
      D:\user\desktop\C++\ODB C++ORM\odb-2.4.0-i686-windows\bin>odb --version
ODB object-relational mapping (ORM) compiler for C++ 2.4.0
Copyright (c) 2009-2015 Code Synthesis Tools CC
This is free software; see the source for copying conditions. There is NO
warranty; not even for MERCHANTABILITY or FITNESS FOR A PARTICULAR PURPOSE.
  2. 编译ODB核心库
    1. 解压libodb-2.4.0到目录D:\user\desktop\C++\ODB C++ORM
    2. 启动VS2005打开压缩包自带的工程文件libodb-2.4.0\libodb-vc8.sln
    3. 添加项目附加路径：鼠标右键：libodb项目 —> 属性 —> C/C++ —> 常规 —> 附加包含目录:...\libodb;..
    4. 编译
    生成lib\odb-d.lib，bin\odb-d-2.4-vc8.dll
  3. 编译sqlite库
    1. 头文件使用sqlite-amalgamation-3080803中的sqlite3.h。 静态链接库需要自己手动生成， lib /out:sqlite3.lib /MACHINE:IX86 /DEF:sqlite3.def 需要注意的是，在编译的时候需要把SQLITE_ENABLE_UNLOCK_NOTIFY宏加上
    生成sqlite.lib
  4. 编译ODB的sqlite库
    生成odb-sqlite-d.lib和odb-sqlite-d-2.3-vc11.dll
