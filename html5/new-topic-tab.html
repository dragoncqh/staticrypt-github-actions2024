Creating New Topics

  1. To open the Resource Manager select one of the following:
    1. View > Resource Manager menu choice.
    2. Insert > New Topic menu choice.
    3. Resource Manager toolbar button.
  2. The Resource Manager opens. Ensure that the New Topic tab is active.
  3. Use the Document Type field to select the type of topic you want to create from the list.
  4. Check the Sample check box if want to use the sample document (for the selected topic type) to create the new topic.

    By default, the template document is used to create a new topic.

  5. Enter a title for the new topic in the Title field.
  6. Enter the path for the folder in which to create the new topic in the Folder field or use the Browse for Folder button () to browse for an existing folder or create a new folder.

    The default folder is the current document directory.

  7. The File Name field is auto-populated based on the topicâ€™s title. You can leave the title alone or enter a different file name.
  8. Use the Insert drop-down list to select the element to insert into your document.

    For unspecialized DITA maps, only the topicref element is available. For specialized DITA maps, specializations of the topicref element are also available. For example, for a DITA BookMap, the chapter and appendix elements are also available.

  9. Press the Insert button.

    Inserts a topic reference to the new topic into the map you are editing at the indicated location and creates the new topic in the folder indicated.

    If the current context in your document is not a valid insertion point, this button is disabled.
