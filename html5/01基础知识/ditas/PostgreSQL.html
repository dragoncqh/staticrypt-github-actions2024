Win 2000 下安装PostgreSQL8.0

功能比MySQL更为强大的PostgreSQL终于release了支持Windows下直接安装的8.0版本。

　　首先说明，PG8.0在Window下的数据库目录最好是NTFS格式，不过我只有FAT32的，将就一下了，不过那样就需要手动 initdb 了。安装时先创建一个用户，一般默认是 postgres，之后我是FAT32的，去掉了那个钩子，跳过数据初始化。 　　安装完毕后，设置 PGHOME=F:PostgreSQL8.0，然后进入Dos，输入 runas /user:postgres “cmd” 切换用户，转到postgresql的 bin 目录，输入 　　initdb –no-locale -D “F:＼PostgreSQL＼8.0＼data” -L “F:＼PostgreSQL＼8.0＼share”

　　如果成功，显示

　　Success. You can now start the database server using: 　　 ““postmaster -D “F:/PostgreSQL/8.0/data” 　　or 　　 ”“pg_ctl -D “F:/PostgreSQL/8.0/data” -l logfile start

　　最后在开始菜单中 start service，PostgreSQL就启动了。运行PGAdmin III，好好体验PostgreSQL吧！
