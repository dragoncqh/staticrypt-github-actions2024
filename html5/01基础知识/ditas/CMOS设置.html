CMOS设置

一，概述

CMOS是主板上的一块可擦写存储器芯片，包含引导系统的BIOS设置以及主板和系统硬件的配置参数。BIOS是主板上一块固化ROM芯片，里面一般厂家都放入了CMOS设置程序和CMOS默认/缺省值(主板配置的最普遍适用的参数)。一般常说的CMOS设置或者BIOS设置都是指通过BIOS程序对CMOS参数进行设置, 是对CPU、内存、硬盘、软盘的参数进行设置，目的是保证机器基本的运行条件。

现在的主板对CPU频率的设置大多可以在BIOS中完成，即所谓“软跳线”。较之过去必须使用主板上的几组拨动开关或跳线来设置，它既提高了安全性，又免去打开机箱仔细查找跳线之劳。当然，如果你使用的是老式主板，还得用跳线来设置。

二，进入CMOS设置

进入CMOS设置:启动计算机。当计算机在启动执行自检时，屏幕的下面一行会出现一个提示，内容大致是：“按Del(F1)键进入CMOS设置”。此时按一下Del键，在自检完成后，就会出现CMOS的设置界面。

三，CMOS的设置内容

通常在主板说明书中都有设置CMOS的详细讲解。如果找不到详细说明书，可以自行浏览条目根据经验判断。由于CMOS设置是在DOS环境下，尚未加载鼠标驱动程序，因此，只能用键盘操作：“←↑→↓”键用来选择项目；“Enter”键用来进入选项；“PageUp/PageDown”用来改变参数；“Esc”键用来退出选项。

无论哪个厂家生产的CMOS，大致都包含如下可设置的内容：

1.Standard CMOS Setup：标准参数设置，包括日期，时间和软、硬盘参数等。 2.BIOS Features Setup：设置一些系统选项。 3.Chipset Features Setup：主板芯片参数设置。 4.Power Management Setup：电源管理设置。 5.PnP/PCI Configuration Setup：即插即用及PCI插件参数设置。 6.Integrated Peripherals：整合外设的设置。 7.其他：硬盘自动检测，系统口令，加载缺省设置，退出等。

不同的主板，BIOS中进入CPU频率设置的条目不尽相同。比如，“升技SE6”主板的条目为“Soft Menu(软跳线菜单)”，而“微星MS-6337”主板的进入条目为“Frequency/Voltage Control(频率/电压控制)”。以升技SE6主板为例：

　　

(1) CPU、内存参数设置。(进入“Soft Menu”条目)

● CPU Operating Speed(CPU工作频率)：

CPU的工作频率=外频×倍频率。对P3 CPU来说，外频可能是100MHz，也可能是133MHz；对于赛扬CPU来说，可能是66MHz或100MHz，这要根据你购买的CPU而定。一般P3铜矿处理器后缀为EB的外频为133MHz，后缀为E的外频为100MHz。另外，在CPU上也能看到66、100或133的字样。AMD的钻龙和速龙/雷鸟目前外频都是100MHz。在这个项目下，按下“Page Up/Page Down”键，可以改变CPU的频率设置，使它与你的CPU一致。比如，你的CPU为P3 733EB，此项目应设置为733(133)。倍频率会自动设置为5.5。

● System Memory Frequency(存储器工作频率)：

设置存储器的工作频率，应设置为外频频率：如CPU外频为133MHz，这里也应设置为133MHz。由于该主板支持内存异步工作，当你的内存条在133MHz工作不稳定时，也可将其设置为100MHz，当然，此时系统效率会降低。

● In-Order Queue Depth(CPU指令序列深度)：

这是涉及P3 CPU工作方式的一项设置。对P3 CPU应设置为4；对赛扬CPU应设置为1。

● CPU Power Supply(CPU电源电压)：

设置为“CPU Default”，使用BIOS缺省设置。

(2) 下面进行硬盘和软驱的设置。进入“Standard CMOS Features”条目，主要针对下面5个项目设置：

● IDE Primary Master(IDE1主设备)；

● IDE Primary Slave(IDE1从设备)；

● IDE Secondary Master(IDE2主设备)；

● IDE Secondary Slave(IDE2从设备)。

上述4项为硬盘与光驱的检测和设置。检测和设置方法基本相同：选中项目，按“Enter”键进入。在后续界面中选择“IDE HDD Auto-Detection(IDE硬盘自动检测)”，并按“Enter”键，检测硬盘或光驱。然后将“IDE……(IDE主、从设备)”和“Access Mode(访问模式)”项目设置为“Auto(自动识别)”。

如果没有第二硬盘和第二光驱，则不需对从设备进行设置。

● Driver A：软驱，设置为1.44MB 3.5in。

至此，我们完成了对CPU、内存、硬盘、软盘的基本参数设置。

另外，某些主板厂商可能还会在CMOS设置中，加入自己特色的设置。比如：对超频CPU的倍频、主频的软跳线设置；对CPU风扇转速、温度的监控设置等。

四，退出CMOS设置

当设置完CMOS后，利用选项“Save ＆ Exit Setup(保存设置退出)”或“Exit Without Saving(不保存设置退出)”可退出CMOS设置，计算机将重新启动。
