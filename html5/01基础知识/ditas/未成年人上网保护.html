未成年人上网保护

概述

未成年人上网保护的过滤软件，又名家长防护(Parental Control)软件，是一种保护儿童及青少年安全浏览互联网的网络安全软件。

主要功能

过滤软件主要有以下功能： 1. 阻挡不良网站（如色情、暴力等不良信息）。 2. 阻挡不良关键词搜寻（如粗口、性用语）。 3. 设定上网时间限制，防止使用者上网成瘾。 4. 监察社交网络、实时通讯、搜寻关键词等记录。 5. 家长阅览使用者浏览记录。

某些过滤软件，更提供自动电子邮件警示功能，提醒家长外出时，留意使用者的网上行为。

软件例子

在商业软件来说，过滤软件一般都是综合型网络安全软件的其中一项功能，如网络安全大师(Internet Security Suite）。但亦有一些单独型过滤软件在网上发售，如Cyber Patrol。 但对于香港家长来说，他们大多数希望提供能够免费使用的过滤软件，故软件制造商亦推出了免费给家长使用的网上过滤软件，例如K9 Web Protection、Windows Live 家长监护服务、Norton Online Family等等。

相关争议

1. 过滤软件可能挡掉子女常去的社交网站，与家长发生争拗。 2. 过滤软件多以过滤英文内容的网站为主，未必能够过滤中文内容的网站。（中国内地研发的过滤软件除外） 3. 子女可能会用不同方式，绕过过滤软件的监控。 4. 中国内地的过滤软件（如绿坝?花季护航），由于配合国情关系，更会过滤政治敏感词汇，以及将浏览者讯息秘密传送给监控当局。 无可否认，要保护儿童及青少年安全使用互联网，过滤软件只是其中一件工具；最重要的是家长与子女的良好沟通，认清网上的各类信息威胁，才能避免双方受到伤害。

一、提醒孩子上网要有社会责任心，注意自己发送的内容不要有失礼的语言；

二、严格控制孩子的网上信件。有的家长甚至掌握孩子电子信箱的密码；

三、要求孩子不要擅自上网，不要看儿童不宜的内容；

四、学生用学校电脑上网时，老师要在场。在家里，家长最好和孩子一起上网。家长要与孩子商量，在孩子同意的基础上规定上网时间，以免孩子沉迷于网络。
