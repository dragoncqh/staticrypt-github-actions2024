HTML入门基础教程

  1. 开始写标签:开始→运行→notepad→填入填入文档类型和开始结尾标签:<!DOCTYPE html><html></html>
  2. 填写HTML文档头部:在<html></html>内填入<head></head>，在<head></head>内填入:
    1. 浏览器顶部标题:<title>HTML入门网站</title>
    2. 网页编码信息:<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    3. CSS样式表信息:<link href="css/styles.css" rel="stylesheet" type="text/css" />
  3. 设计正文布局:在<html></html>内填入<body></body>:HTML文件正文。在<body></body>内填入:
    1. 正文内容:<div id="container"></div>，在<div id="container"></div>内填入:

      1. 头区:<div id="header"></div>

      2. 导航区:<div id="nav"></div>

      3. 特色功能区:<div id="featured"></div>

      4. 内容区:<div id="content"></div>

      5. 清除浮动区:<div class="clear"></div>

    2. 页脚:<div id="footer">
  4. 制作正文头区，记住保存文件为:index.html，在<div id="header"></div>填入:
    1. Logo:最大字号标题1与链接:<h1><a href="#">初学园</a></h1>
    2. phone区:段落与标题6:<div id="phone"><p>+86 12345678</p><h6><h6>如有问题欢迎来电</h6></div>
  5. 制作导航区链接，在<div id="nav"></div>中键入<ul></ul>,并在<ul></ul>中键入:
    1. <li class="active"><a href="#"><span>首页</span></a></li>
    2. <li><a href="aboutus.html"><span>关于我们</span></a></li>，用同样步骤创建:服务/公告/加入我们/联系方式
  6. 制作特色功能区，在<div id="featured"></div>中键入
    1. <ul><li class="ss1"><a href="">特色介绍</a></li></ul>
    2. 插入表单:<form id="quote" action="#" method="post"></form>，在<form></form>中键入:

      1. <h2>快速报价</h2>

      2. <label>您的姓名:</label><input type="text" name="Full Name" />

      3. <label>您的邮件:</label><input type="text" name="Email" />

      4. <label>您感兴趣的产品/服务:</label><textarea cols="35" rows="2"></textarea>

      5. <input class="btn" type="image" src="images/submit_btn.jpg"/>

  7. 制作正文内容区栏目，在<div id="content"></div>中键入:<div class="col"></div>
    1. <h2>关于 <span>初学园</span></h2>
    2. 插入图片:<img src="images/about_smashing_dzine.jpg" alt="关于初学园" />
    3. <p>高级设计师联合育禾网倾情打造，让初学者达到能说能做，知其然知其所以然。</p>
    4. <a href="#" class="readmore">更 多...</a>
  8. 正文内容区按同样步骤创建<div class="col noMargin"><div class="col no Margin fr">
  9. 制作页脚，在<div id="footer"></div>中键入:<p>版权所有 © 2010 YigeChen.com</p><p><a href="#">首页</a> | <a href="#">关于我们</a> | <a href="#">联系我们</a></p>
