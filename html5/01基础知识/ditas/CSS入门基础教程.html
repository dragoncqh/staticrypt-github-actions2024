CSS入门基础教程

  1. 开始写CSS样式:开始→运行→notepad
  2. 设计正文CSS:记住随时保存到css文件夹下文件名为:styles.css，填入:
    1. 初始化内外边距:* { margin:0; padding:0; }
    2. 默认字体:body { font-family: Arial, Helvetica, sans-serif; }
    3. 设置正文区块宽度留白:#container { width: 960px; margin: 0 auto; padding: 0 10px; }
  3. 设计正文头区header:相对定位与宽度:填入:#header { position:relative; width: 100%;}
    1. logo h1样式:#header h1 { display:block; float:left; width:438px; height:90px; margin-top:25px; text-indent:-9999px;}
    2. logo h1 a链接样式:#header h1 a { display: block; height: 100%; width: 100%; }
    3. phone区块样式:#header #phone { height: 47px; position: absolute; top: 41px; right: 0; text-align: right; padding: 5px 50px 0 0; }
    4. 段落样式:#header #phone p { font-size: 20px; }
    5. h6样式:#header #phone h6 { font-size: 11px; }
  4. 设计导航区:
    1. nav区块的初始化去浮定位与宽高:#nav { clear: both; height: 35px; width: 100%;}
    2. UL列表的左对齐定位:#nav ul { float: left; }
    3. 设置nav菜单列表项的格式:#nav ul li { display: block; float: left; height: 35px; list-style-type: none; padding: 0 1px 0 0; }
    4. 设置nav菜单项的链接格式:#nav ul li a { color: #3f3f3f; display: block; text-decoration: none; font-size: 12px; font-weight: bold;text-transform: uppercase; height: 100%; line-height: 35px; padding: 0 0 0 18px; }
    5. 设置nav菜单项的链接span格式:#nav li a span { display: block; float: left; height: 100%; padding: 0 18px 0 0; }
    6. 设置nav菜单项的链接的激活与悬停格式::#nav li a:hover, #nav li.active a { color: #fff; cursor: pointer; text-decoration: none; }
  5. 设置featured特色项目区的格式:
    1. 宽高与留白:#featured { margin: 45px 0 0; background: #fff; padding: 10px; height: 340px; width: 940px; }
    2. 项目列表的左对齐定位:#featured ul { float: left; }
    3. 项目列表项的格式:#featured ul li { float: left; list-style-type: none; display: block; width: 630px; }
    4. 设置项目列表项的链接格式:#featured ul li a { display: block; height: 340px; text-indent: -9999px; }
  6. 设置quick quote快速报价区
    1. 设置表单区格式:form#quote { margin: 0px; padding: 20px; float: right; width: 260px; height: 300px; }
    2. 设置表单区h2标题格式form#quote h2 { font: normal 24px "Trebuchet MS", Arial, Helvetica, sans-serif; color: #fff; text-decoration: none;margin-bottom: 18px; }
    3. 设置标签格式:form#quote label { font-weight: bold; color: #fff; font-size: 12px; display: block; float: left;width: 100%; }
    4. 设置文本框格式:form#quote input, form#quote textarea { background-color: #fff; border:1px solid #ddd; color: #666; float: left; font: 12px Arial, Helvetica, sans-serif; margin: 5px 0 8px; padding: 8px; width: 240px; -webkit-border-radius: 3px; -moz-border-radius: 3px; }
    5. 大文本框格式:form#quote textarea {height: 45px; }
    6. 设置名为.btn的按钮格式，隐藏边框:form#quote input.btn { width: auto; border: none; padding: 0; margin-top: 0; }
  7. 设置Content区格式
    1. Content区格式初始化，去浮动:#content { margin: 80px 0 0; clear: both; font-size: 12px; color: #767676; }
    2. 设置.col:#content .col { float: left; width: 258px; height: 153px; border: 1px solid #CCC; padding: 20px; margin-right: 30px; }
    3. 设置.col中的h2标题:#content .col h2 { font: normal 24px "Trebuchet MS", Arial, Helvetica, sans-serif; color: #b47825;text-decoration: none; margin-bottom: 18px; }
    4. 设置.col中的h2标题span变色区字体颜色:#content .col h2 span { color: #2f2f2f; }
    5. 设置.col中的图片格式:#content .col img { border: solid 1px #d8d8d8; padding: 2px; float: right; margin-left: 10px; margin-bottom: 10px; }
    6. 设置.col段落间距:#content .col p { margin-bottom: 20px; line-height: 17px; }
    7. 设置.col中”更多”的链接格式:#content .col a.readmore { font-weight: bold; color: #252525; text-decoration: underline;}
    8. 设置.col中”更多”的悬停格式:#content .col a:hover.readmore { text-decoration: none; }
    9. 三种特殊样式:外边距全为0:#content .col.noMargin { margin: 0; }
    10. 右对齐定位:#content .col.fr { margin: 0; float: right; }
    11. 清除浮动:.clear { clear: both; }
  8. 设置footer块格式:#footer { clear: both; height: 100px; width: 100%; margin-top: 40px; } #footer p{ margin-left: 350px; }
