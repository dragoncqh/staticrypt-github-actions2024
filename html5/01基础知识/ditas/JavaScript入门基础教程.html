JavaScript入门基础教程

  1. 制作图片:使用Photoshop,在images目录下仿照fp-01.jpg创建两个图fp-02.jpg和fp-03.jpg
  2. 修改style.css:填入:
    1. 设置项目列表项ss2的背景图#featured ul li.ss2 { background: url(../images/fp-02.jpg) no-repeat; }
    2. 设置项目列表项ss3的背景图#featured ul li.ss3 { background: url(../images/fp-03.jpg) no-repeat; }
  3. 修改index.html:加入
    1. script文件:包含预加载但不直接执行的外部JS脚本文件:在</head>之前填入:<script src="js/slide.js" type="text/javascript"></script>
    2. 幻灯图片列表项及其样式:把<div id="featured"><ul></ul>内的<li class="ssl><a href="">特色介绍</a></li>替换为:<li class="mySlides fade ss1"><a href="">特色介绍 1/3</a></li> <li class="mySlides fade ss2"><a href="">特色介绍 2/3</a></li> <li class="mySlides fade ss3"><a href="">特色介绍 3/3</a></li>
    3. script代码:直接包含页面加载后会直接执行的JS脚本代码:须在页面最后</body>之前填入:<script>showSlidesAuto();</script>
  4. 开始写JS脚本文件:开始→运行→notepad→记住随时保存到js文件夹下文件名为:slide.js
  5. 开始写首页幻灯片脚本:变量slideIndex声明:当前图片编号:var slideIndex = 0;
  6. 自动显示幻灯函数showSlidesAuto:function showSlidesAuto(){ }，在{ }内填入:
    1. 变量i声明:图片编号:var i;
    2. 变量slides声明:所有图片集合:var slides = document.getElementsByClassName("mySlides");
    3. 循环语句:隐藏所有图片for (i = 0; i < slides.length; i++){slides[i].style.display = "none"; }
    4. slideIndex自加1:slideIndex++;
    5. slideIndex超过集合图片数时，再重新从1开始:if(slideIndex> slides.length){slideIndex = 1}
    6. 显示当前幻灯图片:slides[slideIndex-1].style.display = "block";
    7. 每2秒轮流显示下一图片:setTimeout(showSlidesAuto, 2000);
      保存slide.js，刷新index.html，即可看到首页幻灯片开始自动滚动
  7. 修改index.html:完善幻灯片功能，在<div id="content">之前加入:
    1. 上一张按钮:<a class="prev" onclick="plusSlides(-1)">&#10094;</a>
    2. 下一张按钮:<a class="next" onclick="plusSlides(1)">&#10095;</a>
    3. 三个图片切换点指示器按钮:<br> <div style="text-align:center"> <span class="dot" onclick="currentSlide(1)"></span> <span class="dot" onclick="currentSlide(2)"></span> <span class="dot" onclick="currentSlide(3)"></span> </div>
    4. 修改script代码:把</body>之前的<script>showSlidesAuto();</script>替换为:<script>showSlides(slideIndex);//showSlidesAuto();</script>
  8. 修改style.css:填入:
    1. 上一张按钮:.mySlides {display:none}
    2. 淡入淡出动画特效:.fade { -webkit-animation-name:fade; -webkit-animation-duration:1.5s; animation-name:fade; animation-duration:1.5s; }
    3. 淡入淡出动画特效:@-webkit-keyframes fade { from {opacity: .4} to {opacity: 1} }
    4. 淡入淡出动画特效:@keyframes fade { from {opacity: .4} to {opacity: 1} }
    5. 上一张下一张按钮样式:.prev, .next { cursor: pointer; position: absolute; top: 330px; width: auto; padding: 16px; margin-top: -22px; color: grey; font-weight: bold; font-size: 18px; transition: 0.6s ease; border-radius: 0 3px 3px 0; }
    6. 上一张按钮左边距:.prev { left:145px; }
    7. 下一张按钮左边距和样式:.next { left:765px; border-radius: 3px 0 0 3px; }
    8. 上一张下一张按钮悬样式:.prev:hover, .next:hover { background-color: rgba(0,0,0,0.8); }
    9. 三个图片切换点指示器按钮样式:.dot { cursor:pointer; height: 13px; width: 13px; margin: 0 2px; background-color: #bbb; border-radius: 50%; display: inline-block; transition: background-color 0.6s ease; }
    10. 小屏幕显示优化:@media only screen and (max-width: 300px) { .prev, .next,.text {font-size: 11px} }
  9. 修改slide.js:填入:
    1. 上一张下一张图片函数plusSlides:function plusSlides(n){showSlides(slideIndex += n);}
    2. 显示指定图片函数currentSlide:function currentSlide(n){showSlides(slideIndex = n); }
    3. 自动显示幻灯函数showSlides:function showSlides(n){ }→在{ }内填入以下所有:
    4. 变量i声明:图片编号:var i;
    5. 变量slides声明:所有图片集合:var slides = document.getElementsByClassName("mySlides");
    6. 变量dots声明:所有切换点集合:var dots = document.getElementsByClassName("dot");
    7. n超过集合图片数时，slideIndex再重新从1开始:if (slideIndex> slides.length) {slideIndex = 1}
    8. n<=0时，slideIndex取最后一张图片编号:if (n < 1) {slideIndex = slides.length}
    9. 循环语句:隐藏所有图片for (i = 0; i < slides.length; i++){slides[i].style.display = "none"; }
    10. 循环语句:反激活所有切换点for (i = 0; i < dots.length; i++) { dots[i].className = dots[i].className.replace(" active", ""); }
    11. 显示当前幻灯图片:slides[slideIndex-1].style.display = "block";
    12. 激活当前切换点:dots[slideIndex-1].className += " active";
    保存slide.js，刷新index.html，即可看到首页幻灯片出现上一张、下一张和切换点按钮
