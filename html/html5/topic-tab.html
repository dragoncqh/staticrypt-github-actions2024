Inserting Topics

The Topic tab enables you to insert topic references into DITA maps. This tab is available in the Resource Manager when you are editing a map or a specialization of a map.

The Topic tab has two different states for inserting topic references:
  • local documents
  • documents on the Internet

Inserting topic references to local documents

  1. Select the Insert > Topic Reference menu choice.
  2. The Resource Manager opens with the Topic tab active.
  3. Browse for the document you want to insert.
    You can also:

    • select multiple files and insert references to all of them at the same time.

    • select a folder and insert references to all of the topics in that folder at the same time.

  4. Use the Insert drop-down list to select the element to insert into your document.

    For unspecialized DITA maps, only the topicref element is available. For specialized DITA maps, specializations of the topicref element are also available. For example, for a DITA BookMap, the chapter and appendix elements are also available.

  5. Press the Insert button.

    This inserts a topic reference to the document into the map you are editing at the indicated location.

    If the current context in your document is not a valid insertion point, this button is disabled.

Inserting topic references to documents on the Internet

For inserting topic references to a document on the Internet, the Topic tab contains the following options:
  1. Select Insert > Topic Reference menu choice.
  2. The Resource Manager opens with the Topic tab active.
  3. Press the Browse the Web button on the Topic tab.

    This changes the state for inserting references to the Internet and invokes a web browser.

    If you already have a web browser open, the browser is made the active window. If you do not have a browser open, your default web browser is invoked.

  4. Use the web browser to find the URL you want to insert and then go back to the Topic tab.
  5. The URL field displays the URL to be inserted for your topic reference.

    By default, this field displays the URL for the document currently open in your active web browser. The URL field automatically updates when you browse to a new web page and move the focus back to the Resource Manager. You can also edit this field.

  6. Use the Insert drop-down list to select the element to insert into your document.

    For unspecialized DITA maps, only the topicref element is available. For specialized DITA maps, specializations of the topicref element are also available. For example, for a DITA BookMap, the chapter and appendix elements are also available.

  7. Press the Insert button.

    This inserts a topic reference to the content into the map you are editing at the indicated location.

    If the current context in your document is not a valid insertion point, this button is disabled.
